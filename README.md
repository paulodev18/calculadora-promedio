# Proyecto Java - 3 Paradigmas de Programación

## 📚 Descripción
Este proyecto demuestra la implementación de tres paradigmas de programación en Java para calcular promedios de calificaciones.

## 🖥️ Paradigmas Implementados

### 1. Estructurado (`Main.java`)
- Variables y secuencias
- Condicionales para determinar aprobación
- Enfoque procedural

### 2. Imperativo (`MainImperativo.java`)
- Bucles `for` para iterar sobre arreglos
- Acumulación de sumas
- Conteo de aprobados

### 3. Orientado a Objetos (`Estudiante.java` + `MainPOO.java`)
- Clase `Estudiante` con atributos privados
- Métodos: `calcularPromedio()`, `estaAprobado()`, `mostrarInfo()`
- Encapsulamiento y reutilización

## 🚀 Cómo ejecutar

```bash
# Compilar
javac src/*.java

# Ejecutar programa estructurado
java -cp src Main

# Ejecutar programa imperativo
java -cp src MainImperativo

# Ejecutar programa POO
java -cp src MainPOO
